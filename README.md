## View slides locally

First, ensure you have the following installed:

0. [Node.js](http://nodejs.org)
0. `git clone https://github.com/lgersman/enterjs2015-polymer_es6.git`
0. `npm install && node_modules/.bin/bower install`
0. `npm run serve`
